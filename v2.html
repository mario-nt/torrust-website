Modern BitTorrent index and tracker

Torrust is an open source project that aims to provide you with the tools you need to host your own torrent indexes with ease.

index

v 3.0.0

The Index is a feature-rich torrent indexing site split between an API (backend) built with Rust, and our reference web app that consumes the API, built with Vue 3.

tracker

v 3.0.0

A lightweight Rust-based BitTorrent tracker for efficient torrent serving to many peers with high performance, robustness, security, and community support.
